Gmail・Discord・Slack の未読
タブを開かずに確認する方法

2026.07.24 · Liquid Block Inc. · 読了 4分

仕事中、こんな経験はありませんか？

  • 「Gmailに新着きてるかな…」とタブを切り替えて確認 → 集中が途切れる
  • 「Slackで誰か呼んでるかも」と気になってチラ見 → 別のチャンネルに気を取られて10分ロス
  • 「Discordの通知音は聞こえたけど、何件溜まってるか分からない」

タブを開かなくても未読がわかれば、作業の流れを止めずに済むのに——。

LB TAB Layers の通知バッジ機能は、まさにこの問題を解決します。

🔔 通知バッジとは？

通知バッジは、Gmail や Slack などの連絡系サービスをフォルダに入れておくだけで、未読メッセージ数をサイドパネル上にリアルタイムで赤バッジ表示する機能です。

スマートフォンのアプリアイコンに表示される赤い数字バッジと同じ感覚で、ブラウザのサイドパネルから未読状況をひと目で把握できます。

📮 対応サービス一覧

以下のような連絡系・通知系サービスで未読バッジが自動表示されます：

✉️ メール

  • Gmail — 未読メール数をバッジ表示。タブを開かなくても「今3件溜まってるな」がわかる
  • Outlook（Web版） — Microsoft 365 ユーザーにも対応

💬 ビジネスチャット

  • Slack — 未読メッセージ・メンション数を表示。チャンネルを巡回しなくても確認OK
  • ChatWork — 日本のビジネスシーンで多用されるChatWorkの未読も一目瞭然
  • Microsoft Teams（Web版） — チームの通知を見逃さない

🎮 コミュニティ・SNS

  • Discord — サーバーごとの未読・メンション数をバッジで表示。ゲーム中でもコミュニティの動きがわかる
  • Facebook / Messenger — 友達からのメッセージや通知の未読数
  • X（旧Twitter） — DMやリプライの未読通知

🎬 メディア・その他

  • YouTube — チャンネル登録先の新着動画通知
  • LINE（Web版） — ブラウザ版LINEの未読メッセージ
  • WhatsApp Web — 海外の連絡先との未読も表示

💡 仕組み：なぜバッジが表示されるのか

通知バッジは、各Webサービスのページタイトルに含まれる 未読数の数字を自動で検出して表示しています。

たとえば Gmail では、未読メールがあるとブラウザのタブタイトルが 受信トレイ (3) - Gmail のように変わります。LB TAB Layers はこの (3) を読み取って赤バッジに変換しているのです。

そのため、追加の設定やAPI連携は一切不要。フォルダにサービスのタブを入れておくだけで、自動的にバッジが表示されます。

🎯 活用シーン

シーン1：集中作業中の「チラ見」を最小化

コードを書いている最中、企画書を書いている最中、デザインに集中している最中——。サイドパネルの端に赤バッジが見えたら「あとで対応しよう」、バッジがなければ「安心して作業を続けよう」。タブを切り替える必要がないので、集中力の損失が最小限です。

シーン2：複数サービスの通知を一箇所で管理

Gmail・Slack・ChatWork・Discord——仕事で使う連絡ツールが多い人ほど、通知バッジの恩恵は大きくなります。

たとえば「連絡・通知」というフォルダを作って、以下のタブをまとめて入れておくだけ：

  • 📧 Gmail
  • 💬 Slack
  • 💬 ChatWork
  • 🎮 Discord
  • 📘 Facebook Messenger

フォルダ横にバッジが表示されるので、どのサービスに未読があるかが サイドパネルを一目見るだけでわかります。

シーン3：「タブを開きっぱなし」から卒業

従来は「Gmail の未読が気になるからタブを閉じられない」「Slack のタブは常に開いておかないと不安」という理由で、タブを開きっぱなしにしていたかもしれません。

通知バッジがあれば、タブを閉じてもバッジで未読がわかるので、安心してタブを閉じてメモリを節約できます。必要なときにフォルダからワンクリックで復元すればOKです。

⚙️ 設定方法（かんたん3ステップ）

  1. フォルダを作成 — サイドパネルで「連絡ツール」や「通知」などの名前でフォルダを作成
  2. タブを追加 — Gmail・Slack・Discord などのタブをフォルダに追加（＋ボタンまたは右クリック保存）
  3. 自動でバッジ表示 — 未読があれば赤バッジが自動的に表示されます。設定完了！

特別な設定画面を開く必要はありません。フォルダに入れるだけで機能します。

🏆 サイドパネルにここまで執着した拡張は、他にない

Chrome の「サイドパネル」は 2022 年に導入された比較的新しい機能です。多くのタブ管理拡張機能は別タブやポップアップでUIを表示しますが、LB TAB Layers はサイドパネルを作業ハブとしてフル活用することに徹底的にこだわっています。

サイドパネルに搭載されている機能を並べると、その密度がわかります：

  • 📂 レイヤーグループ管理 — フォルダでタブを整理・一括開閉
  • 🔔 通知バッジ — Gmail・Slack・Discord の未読をリアルタイム表示
  • 📌 バンパーピンバー — よく使うサイトを最大10個常駐
  • 🔍 AI 検索 — フォルダを横断してタブを意味的に検索
  • 📰 RSS タイムライン — ニュースフィードをカード表示で受信
  • ⏱ ポモドーロタイマー ＆ 時計 — 集中と休憩のリズム管理
  • 📋 単独タブ ＆ 履歴 — 未整理タブと今日の閲覧履歴

これだけの機能が、作業画面を離れずに横のパネルで全て操作できる。しかも左右の配置切替や 40 種類のテーマでデザインも自由。

ポップアップ型のタブマネージャーは「開いて → 操作して → 閉じる」ステップが必要ですが、サイドパネル常駐型なら常にそこにある。この「ながら管理」の感覚が、通知バッジと組み合わさることで真価を発揮します。

タブ管理だけでなく、通知確認・情報収集・時間管理までサイドパネル1つで完結する——それが LB TAB Layers の「サイドパネルへの執着」です。

まとめ

LB TAB Layers の通知バッジ機能は、「タブを開かずに未読がわかる」というシンプルな体験を提供します。

Gmail・Slack・Discord・Facebook・YouTube・ChatWork——普段使っている連絡系サービスをフォルダに入れておくだけで、サイドパネルが通知ダッシュボードに変わります。

集中を妨げずに通知を管理したい方は、ぜひ試してみてください。
