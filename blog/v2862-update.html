v2.8.62 アップデート
タブUI統合・操作性の進化

2026.07.23 · Liquid Block Inc. · 読了 3分

LB TAB Layers v2.8.62 をリリースしました。本バージョンでは、サイドパネル下部のUI構造を大幅に刷新し、操作性とスペース効率を向上させています。

🗂️ 単独タブ＋本日の履歴 → タブUI統合

これまで2行のセクションヘッダーで分かれていた「単独タブ」と「本日の履歴」を、1行のタブバーに統合しました。

  • Chrome風のタブデザイン — アクティブタブにはアクセントカラーの下線＋背景ハイライト
  • タブ間にセパレーターラインを配置し、視覚的な区切りを明確化
  • 単独タブのアクションボタン（ゴミ箱・自動振分け・新規フォルダ）はタブバー内に統合
  • 「本日の履歴」タブ選択時、アクションボタンはスペースを保持したまま非表示化（レイアウトシフトなし）

この変更により、ヘッダー1行分の画面スペースを節約し、より多くのタブを一度に表示できるようになりました。

▶ ワンクリック折りたたみ

タブバーの先頭にアクセントカラーのシェブロンボタン（▼/▶）を追加しました。

  • クリックするとコンテンツエリアが折りたたまれ、タブバーだけの表示に
  • もう一度クリックすると、アクティブなパネルが再表示
  • グループリストをフル画面で使いたい時に便利

⏱ タイマーの操作性改善

ピンバーのタイマー機能を改善しました。

  • タイマー未起動時に「⏱ タイマー」のテキストラベルを表示 — アイコンだけでは分かりにくかったフィードバックを反映
  • 実行中・一時停止中のスタート/停止ボタンの視認性を向上
  • タイマー設定ポップアップの表示位置を最適化（position:fixedで他の要素と干渉しない配置に）

♿ アクセシビリティ強化

Microsoft Edge Tools のアクセシビリティ検査結果を反映し、以下を修正しました。

  • select要素 — ダッシュボードのソートメニューに aria-label を追加
  • ボタン要素 — アイコンのみのボタンにスクリーンリーダー用テキストを追加

スクリーンリーダーをご利用の方にも、より快適にお使いいただけるようになりました。

アップデート方法

Chrome Web Store から自動更新されます。手動で更新する場合は chrome://extensions を開き、「更新」ボタンをクリックしてください。

🚀 Chrome に無料で追加するChrome Web Store → 無料インストール（30秒で完了）
