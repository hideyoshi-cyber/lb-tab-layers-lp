v2.8.61 アップデート
8言語完全対応・CSS設計刷新

2026.07.22 · Liquid Block Inc. · 読了 4分

LB TAB Layers v2.8.61 をリリースしました。本バージョンでは、6つの新言語の追加によるグローバル対応、CSS設計の全面的なリファクタリング、そして右クリックメニューからのフォルダ直接保存など、品質と使いやすさを大きく向上させています。

🌐 8言語完全対応 — 世界のユーザーへ

これまで日本語と英語の2言語対応でしたが、v2.8.61では以下の6言語を追加し、計8言語に完全対応しました。

  • 🇰🇷 韓国語（한국어）
  • 🇹🇼 台湾中国語（繁體中文）
  • 🇩🇪 ドイツ語（Deutsch）
  • 🇫🇷 フランス語（Français）
  • 🇪🇸 スペイン語（Español）
  • 🇵🇹 ポルトガル語（Português）

単なるボタンラベルの翻訳にとどまらず、ヘルプガイド全4ステップ、チュートリアル、PRO/アップグレード画面、週間レポート、フィードバックフォーム、エラーメッセージまで、全369キー中367キー（99.5%）を翻訳しています。未翻訳の2キーはブランド名（「LB TAB Layers」「PRODUCED BY」）のため翻訳不要です。

ブラウザの言語設定に自動追従するため、ユーザーは何も設定する必要がありません。

📐 CSSリファクタリング — 保守性の大幅向上

これまで1ファイル 3,682行 に集約されていた sidepanel.css を、8つの論理モジュールに分割しました。

  • themes.css — 40テーマのCSS変数定義
  • base.css — フォント・リセット・ユーティリティ
  • layout.css — ヘッダー・検索・セクション配置
  • components.css — グループカード・タブ行・ピンバー
  • panels.css — 設定・ヘルプ・モーダル
  • features.css — AI検索・オンボーディング・RSS
  • upgrade.css — PRO・トライアル・レポート
  • animations.css — アニメーション・トランジション

この変更はユーザーから見た見た目や動作には影響しませんが、開発チームにとって「どのファイルのどこを触ればいいか」が明確になり、今後の機能追加やバグ修正のスピードが大幅に向上します。

📂 右クリックでフォルダに直接保存

単独タブ（Ungrouped）や本日の履歴にあるサイトを右クリックすると、コンテキストメニューに「フォルダに保存」オプションが表示されるようになりました。任意のグループフォルダを選ぶだけで、ドラッグ操作なしで素早くタブを整理できます。

🔤 フォント＆密度カスタマイズ

設定パネルに新しい「テキスト＆密度」セクションを追加しました。

  • フォントサイズ — スライダーで自由に調整
  • タブ行の高さ — コンパクト〜ゆったりまで
  • フォルダヘッダーの太さ — Thin / Normal / Bold
  • フォルダ名の色調 — Default / Medium / Light

小さな画面でも大きなモニターでも、自分好みの情報密度でタブを管理できます。

🔧 Stickyヘッダー修正

単独タブセクションのヘッダーがスクロール時に隠れてしまう問題を修正しました。原因は親要素の overflow: hidden が position: sticky を無効化していたためで、overflow: visible に変更することで解決しました。

📊 ストアリスティングの修正

Free版とPRO版の機能比較表を、コード上の実際の制限と一致するよう修正しました。

  • オーバーレイモード → 実際はPRO/Trial限定（Free版から除外）
  • ダッシュボード → PRO版の特典として追記
  • ★ランク/タグ → Free版は10個まで、PRO版は無制限と明記
