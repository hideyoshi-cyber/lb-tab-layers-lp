브라우징을 혁신하는
TAB Layers 기술.

일상 작업 속도와 브라우저 편의성을 획기적으로 높이는 세련된 기능을 소개합니다.

작업 맥락을 전환하는
「레이어형」탭 정리

프로젝트, 클라이언트, 용도(리서치, 회계, SNS 등)별로 탭을 「레이어 그룹」으로 정리·관리합니다.
원클릭으로 그룹 내 모든 탭을 일괄 열고, 작업이 끝나면 한 번에 닫아 메모리를 해방합니다.

  • 다중 탭 일괄 복원 & 일괄 닫기
  • 드래그 앤 드롭으로 직관적 정렬
  • 하위 그룹(폴더 계층화) 완벽 지원

절대 사라지지 않는 안심감
북마크 양방향 자동 동기화

다른 탭 관리 도구와 달리, TAB Layers에 저장한 그룹과 탭은 모두 브라우저 기본 북마크와 자동 동기화됩니다.
확장을 삭제하거나, 브라우저가 충돌하거나, PC가 강제 재시작되어도 데이터는 절대 사라지지 않습니다.

  • 확장 삭제 후에도 북마크에 그대로 보존
  • 양방향 동기화: 북마크 편집이 TAB Layers에 자동 반영
  • 서버를 거치지 않아 프라이버시 완벽 보장

의미로 검색·자동 정리
AI 어시스턴트 기능(Beta)

Chrome 내장 AI(Gemini Nano)나 보유한 API 키(Gemini / OpenAI / Claude)를 연결하면 고급 AI 검색이 활성화됩니다.
「출장용 호텔」「다음주 회의 자료」같은 추상적 키워드로 탭을 검색하거나, AI가 열린 탭을 분석해 자동으로 폴더를 분류·명명합니다.

  • 시맨틱 검색: URL이나 제목에 없는 의미를 파악
  • AI 자동 그룹화: 원클릭 자동 카테고리 분류
  • 프라이버시 배려: API 연결은 로컬에서 실행

중요한 것을 눈에 띄게
★ 중요도 랭킹 기능

그룹(폴더)과 개별 탭에 ★1~★5 중요도 랭킹을 부여할 수 있습니다.
중요한 프로젝트나 자주 참조하는 페이지에 ★를 붙여두면 수십 개 그룹에서도 한 눈에 찾을 수 있습니다.

  • 그룹·탭 단위 ★1~★5 랭킹 부여
  • 원클릭 팝업 피커로 빠른 랭킹 변경
  • 대시보드에서 ★랭킹 순 정렬·필터링

사이드 패널과 대시보드 모두에서 사용 가능

폴더를 넘어 찾기
#태그 자유 분류

폴더 계층만으로 부족한 분류 니즈를 「태그」 기능이 채웁니다.
개별 탭에 #디자인, #참고자료, #확인필요 등 자유로운 태그를 여러 개 붙이고, 폴더를 넘나들며 같은 태그의 탭을 필터링합니다.

  • 탭에 여러 태그 자유롭게 추가/삭제
  • 대시보드에서 태그별 필터링
  • 북마크 정리 화면에서도 태그 부여 가능
#디자인 12 #참고자료 8 #확인필요 3

태그를 클릭하면 전체 폴더에서 해당 탭만 표시

대화면에서 스마트하게 정리
기존 북마크 분류·이관

사이드 패널 외에도 전체 화면 「대시보드」 기능을 탑재.
기존 브라우저 북마크를 한눈에 보고, 클릭이나 드롭다운 폴더 선택으로 TAB Layers 그룹 폴더에 직관적으로 분류·이관합니다.

  • 북마크 바와 기타 북마크의 스마트 일괄 정리
  • 폴더 선택 리스트로 목표 레이어 폴더에 빠른 이관
  • 넓은 화면에서 탭·그룹·태그 병렬 정리
