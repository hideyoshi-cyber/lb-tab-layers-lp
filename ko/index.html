자주 묻는 질문

확장을 삭제하면 데이터는 어떻게 되나요?

탭은 Chrome 북마크로 남으므로 확장을 삭제해도 링크는 사라지지 않습니다. 재설치하면 그룹과 탭이 복원됩니다.
※ 테마·색상·태그 설정은 로컬 저장이므로 삭제 시 초기화됩니다.

PRO는 구독인가요?

아닙니다. 일회성 구매입니다. 현재 한정 세일 $4.99 (정가 $9.99). 한 번 결제하면 영구적으로 사용 가능합니다.

어떤 브라우저에서 사용할 수 있나요?

Google Chrome 114 이상, Microsoft Edge (Windows / Mac / Linux / ChromeOS).

PC 재시작 후 탭을 복원할 수 있나요?

네. LB TAB Layers로 관리한 탭은 Chrome 북마크로 저장되어 있으므로, 재시작·충돌·강제 종료 후에도 사이드 패널에서 원클릭으로 복원 가능합니다.
